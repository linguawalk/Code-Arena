# Code Arena

"언어를 배우는 것이 아니라 생각하는 방법을 배운다"

프로그래밍 개념을 퍼즐 형태로 학습하는 플랫폼입니다. 컴퓨팅 사고와 C 언어로 구성된 **공통 기초 구간**을 모두가 거친 뒤, 관심사에 따라 **시스템/저수준, 객체지향/엔터프라이즈, 스크립트/실무, 레거시/역사** 4개 카테고리 트랙 중 원하는 트랙으로 분기해 학습합니다.

## 구조

```
공통 기초 (컴퓨터 사고 → C 언어)
        │
        ▼
   [트랙 선택]
        │
   ┌────┼────┬────┐
   ▼    ▼    ▼    ▼
 시스템 OOP  스크립트 레거시
```

자세한 커리큘럼 설계는 [`curriculum.md`](./curriculum.md)를 참고하세요.

## 폴더 구조

- `foundation/` — 공통 기초 구간 콘텐츠 (컴퓨터 사고, C 온보딩)
- `content/{category}/{language}/` — 트랙별 언어 콘텐츠
- `modules/` — 트랙 무관 공통 선택 모듈 (자료구조·알고리즘)
- `schema/` — 문제 JSON 스키마 정의
- `meta/` — 언어별 진행 상황 트래킹

## 현재 상태

기초 구간 및 4개 트랙 대표 언어(C, Visual Basic, Python, BASIC 초기형)의 Level 1 콘텐츠 스키마를 검증 중입니다.

## 기술 스택

GitHub + Vercel 기반 정적 사이트 (vanilla HTML/CSS/JS). 향후 트래픽 증가 시 Pyodide(브라우저 내 Python 실행), 서버 사이드 채점 엔진 도입 예정.
